# base64-coder
simple base64 coder/decoder written in Golang

### :computer_mouse: [Releases](https://github.com/Sagleft/base64-coder/releases)

### usage example

Encode:

```bash
./b64 encode from.txt to.dat
```

Decode:

```bash
./b64 encode to.dat from.txt
```

### build

```bash
mkdir build
bash cross_build.sh
```

or

```bash
go build
```

### install (linux)

```bash
mv b64 /usr/bin/
```
